# Kamis
# 🎂 Surpresa de Aniversário - Para Kamis

Este é um projeto simples e rápido desenvolvido para celebrar o aniversário de uma amiga especial (**Kamis**). A ideia foi criar algo interativo e carinhoso para acompanhar o presente principal.

## ✨ O que é?
Uma carta digital interativa dentro de um envelope animado. Ao clicar no envelope:
- A aba abre.
- O papel da carta sobe com uma animação suave.
- Ocorre uma chuva de confetes coloridos.
- O texto possui áreas de destaque e elementos divertidos (como textos riscados e marca-texto).

## 🛠️ Tecnologias Utilizadas
Como o objetivo era rapidez e simplicidade, o projeto foi construído com o básico da web:
- **HTML5**: Estrutura da carta e do envelope.
- **CSS3**: Animações (`keyframes`, `transitions`) e design responsivo.
- **JavaScript**: Lógica de abertura e disparo dos confetes.
- **Canvas-confetti**: Uma biblioteca leve para o efeito visual de festa.

## 🚀 Como visualizar
O projeto está hospedado na Vercel e pode ser acessado pelo link enviado diretamente para a aniversariante.
